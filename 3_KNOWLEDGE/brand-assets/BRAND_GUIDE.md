# SEOSONA Brand/UI Guide

## Direction

Modern Google Marketing Agency: tin cậy, chuyên gia, tăng trưởng, dữ liệu, rõ ràng.

## Colors

- Primary Blue: #155EEF
- Primary Dark: #1D4ED8
- Cyan: #06B6D4
- Violet: #7C3AED
- Text/Navy: #0F172A
- Muted: #64748B
- Background: #F8FAFC
- Border: #E2E8F0

## UI principles

- Clean SaaS/agency layout
- Large whitespace
- Card-based sections
- Strong CTA
- Real proof: case study, logos, expert profiles
- Lightweight animation only when needed
- No legacy shortcode/form output

## Core components

- Header + mega menu
- Footer
- Hero
- Client logos
- Service cards
- Case study cards
- Testimonial
- CTA block
- Contact form
- Breadcrumb
- Table of contents
- Related posts
- FAQ accordion
